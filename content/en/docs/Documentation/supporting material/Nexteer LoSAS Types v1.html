DESCRIPTION:
Steering angle sensing data on the vehicle bus is frequently another safety-critical function of EPS and some OEM programs are beginning to request redundancy to avoid loss of this signal. Although EPS does not require this signal to provide assist, loss of this signal may create a safety hazard in some vehicle functions. This sheet describes the type of loss of steering angle sensing (LoSAS) mitigation solutions offered by Nexteer.
